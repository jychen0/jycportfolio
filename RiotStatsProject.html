Riot API Stat Tracking

Track your or opponent stats on League of Legends

Description

This project utilizes the Riot Games API with python to find the stats of a inputed Riot ID. It will average out common stats for a user defined amount of games. There are plans to create an app from this with a GUI and an easier way to save the data.
This game was created for my CSCI120 class as the final project. It was utlized by the Clark University League of Legends team and will continue to be developed.
