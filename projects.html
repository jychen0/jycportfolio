Riot Games API Stats Tool

Riot Games API Stats Tool

Individual

A project utilizing the Riot Games API for League of Legends, this tool will request for you to enter a player's ID and tagline, a region, the specific queue you want to search (on summoners rift), and the amount of games you want to search. It then grabs all the data from those games and then puts them into a .csv file that you can upload onto google sheets. After this, it will prompt you to see if you want to see graphs of some of the data or not.

This project was made for my CSCI 120 class and then utilized for the Clark University Esports Team and also my own data tracking

A google sheets template for your own data tracking is a WIP and will be added to the github repository once it is done.

Programmer
