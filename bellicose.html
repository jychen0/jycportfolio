Bellicose

Demonstrating aggression and willingness to fight.

Team: Blake Shea, Sean McDowell, Anthony Pansenko, Liam Quinn

Role: Programmer, UI/UX

Last Updated: April 13, 2024

Link(s): Itch.io link coming soon

Description

Bellicose is a 2D fighting game utilizing a pixel art style and implements typical fighting game mechanics such as different height blocks/attacks, staggering attacks, different special abilities between characters, and more to pit two players against each other in a skill based combat system.
This game was created for the Clark University IGDA Feburary Game Jam with the theme "Retro"

Contributions

- Programmed and created a unique character select UI
- Programmed the timer, health bar, and random background clouds
- Implemented all character animations using a state machine
