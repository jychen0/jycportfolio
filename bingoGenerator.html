Custom Bingo Generator

Create your own bingo board!

Solo Project

Role: Programmer

Last Updated: July 2, 2024

Link(s): github

Description

This project utilizes React.js and CSS to create a site that can generate a custom bingo board by inserting your own spaces. There are small popups in case the space is too detailed to fit inside the bingo square. You are able to click on each individual space to mark it, with the middle space always being a "free space". Future updates could include making it mobile friendly.

Contributions

- Designed the layout for a site that creates custom bingo boards
- Programmed the functionality using React.js
- Learned to connect a .txt file to javascript allowing me to have default values for the bingo board
